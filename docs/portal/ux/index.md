# Portal UX Documentation

UX artifacts for the portal app. Each page has three document types.

## Document Types

| Folder        | Purpose                                                              | Used By                                           |
| ------------- | -------------------------------------------------------------------- | ------------------------------------------------- |
| `flows/`      | User flows — step-by-step interactions                               | `/test:e2e` agent for generating Playwright tests |
| `specs/`      | Detailed specifications — requirements, edge cases, validation rules | Implementation reference                          |
| `wireframes/` | ASCII wireframes — visual layout and component placement             | Design reference                                  |

## Pages

- **login-page** — Authentication flow
- **employee-page** — Employee listing and management
